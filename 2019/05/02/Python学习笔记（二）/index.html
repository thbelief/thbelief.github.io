相对不能修改性

表面上来看，元组不能修改，但是，真的是这样麽？

猜猜结果是什么？

不是说好不变的麽？为什么变了呢？

答案是——元组并没有变，变的是元组内部的列表。指向的list并没有改变，只是其中的内容改变了！
